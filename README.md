# Agent 面试题集

一个面向 AI/LLM 方向求职与面试准备的知识库网站，聚焦 Agent 相关高频问题，覆盖从大模型基础到工程落地的核心主题，适合用于系统复习、面试冲刺和团队内部知识沉淀。

## 在线访问

- 网站地址：[https://qaqnan.github.io/](https://qaqnan.github.io/)

## 主要内容

- `大模型`：Transformer、微调、对齐、生成问题等基础与进阶问题
- `Agent`：架构设计、记忆机制、工具调用、评估与优化
- `RAG`：检索增强生成的流程、索引、召回、重排与评估

## 网站特性

- 三栏阅读布局：左侧导航、中间正文、右侧当前页小标题目录
- 日间/夜间模式切换，提升不同环境下的阅读体验
- 原生导航配置：通过 `_data/navigation.yml` 精确控制页面顺序与展示

## 项目结构

- `index.md`：站点首页
- `_layouts/default.html`：站点主布局模板
- `assets/css/style.css`：全站样式（含主题样式）
- `_data/navigation.yml`：左侧导航配置（顺序与显示入口）
- `大模型.md` / `Agent.md` / `RAG.md`：主要题库内容

## 本地预览

如果你希望在本地预览 GitHub Pages/Jekyll 页面效果，可执行：

```bash
bundle exec jekyll serve
```

然后访问：

- `http://127.0.0.1:4000/`

## 内容维护说明

- 新增页面后，请同步更新 `_data/navigation.yml`，否则不会在左侧导航显示
- 建议使用二级、三级标题（`##` / `###`），便于右侧小标题目录自动生成
- 页面链接统一使用 `.html` 形式，减少跳转与渲染差异
